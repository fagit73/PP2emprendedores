<h3>-Descripción del proyecto</h3>
Este sistema tiene como objetivo digitalizar y optimizar la gestión de reservas de una biblioteca escolar, reemplazando el sistema manual basado en cuadernos.

Permite:

<ul>
    <li>Gestionar reservas de una sala multipropósito</li>
    <li>Evitar superposición de turnos</li>
    <li>Registrar tipo de uso (lectura, audiovisual, extensión cultural)</li>
    <li>Generar reportes para la toma de decisiones institucionales</li>
    <li>Diferenciar roles de usuarios (docente, directivo, administrador)</li>
</ul>

<h3>-Objetivos</h3>

<ul>
    <li>Optimizar la asignación de espacios</li>
    <li>Evitar errores humanos en reservas</li>
    <li>Digitalizar el control de actividades</li>
    <li>Facilitar informes estadísticos</li>
    <li>Mejorar la planificación institucional</li>
</ul>
